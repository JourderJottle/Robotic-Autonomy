# To Load Code
1. Load all three files into the same folder on the laptop
2. Launch Arduino IDE
3. Load "ChassisArduinoWithShield.ino" onto the Arduino

# To Tune Speed
Tweak "speed_scalar" to change the speed of both motors

# Difference from Arduinos with Separate Controllres
- Different pins for the motors
- All LEDs excpet for BuiltIn are disabled to avoid conflict with other pins on the shield
